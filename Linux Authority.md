## chmod 400? 755?

수업을 듣다가 `sudo chmod 755 initial_ec2.sh` 가 나와서 번호의 의미가 궁금해짐.
리눅스는 `chmod` 로 파일 또는 경로에 아래와 같이 권한을 설정할 수 있음.

```
sudo -s // root 권한 얻기
chmod -R 777 filename
```

또한 기본적으로 권한 값을 갖게 되는데, 이들이 숫자로 표시되는 것. 파일을 서버에 전송하기 전에도 역시 전송될 파일에 권한이 부여됨.

간단하게 아래와 같다.
```
읽기 reading -4
쓰기 writing -2
실행 executing -1

1번 : 소유자 권한
2번 : 그룹 사용자 권한
3번 : 기타 사용자 권한
```

ex1) 777 -> 모든 사용자가 읽고, 쓰고, 실행이 가능하다.

ex2) 755 -> 소유자만 읽고, 쓰고, 실행이 가능하며 / 그 외 사용자는 읽기, 실행만 가능하다.
